Final Year Project

PIDS — Proactive Intrusion Detection System

Oct 2025 – Present

A full-stack intrusion detection system fusing a four-stage ML pipeline with a local LLM behaviour engine to catch network attacks in real time. Traffic flows through a binary classifier, a 14-class attack identifier, a connection-rate detector for volumetric attacks, and an Ollama-hosted LLM that reasons over C2 beaconing, scanning, SQL injection, and exfiltration patterns.

  • Trained on the full CICIDS2018 dataset — 15.7M rows / 6 GB — with GPU-accelerated XGBoost at 98.24% accuracy
  • 31 statistical features computed live from raw packets via Scapy; user-selectable XGBoost / LightGBM / DNN / GNN engines
  • Extended with E-GraphSAGE + GATv2 GNN engine, lifting Infiltration F1 by 2.34× over the DNN baseline
  • React 18 + Material-UI analyst dashboard with live threats, LLM re-analysis, and automated PDF reports
  • JWT-based RBAC (5 roles, HMAC-SHA256), full password policy, JSON audit logging & a 40-issue security audit
  • Validated against 10 live attacks using a custom attack generator and listener, confirming detection across major categories
